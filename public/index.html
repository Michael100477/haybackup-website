Questions, answered

Does it work on Windows Server?

Yes — any modern Windows PC or Server. It uses only built-in Windows tools, so there's nothing third-party to install or license.

Are the disk images really bootable?

Yes. They're block-level .vhdx images you can mount to recover individual files, or attach to a Hyper-V Generation 2 VM and boot.

Where do my backups go?

Wherever you choose — a local drive, a USB drive, or a network share (NAS). Your data never passes through us.

What happens if my subscription lapses?

HayBackup keeps running and only shows reminders — it never interrupts your backups. Every install also includes a 7-day grace period.

Can I back up other computers?

Yes. Install a lightweight agent on any Windows PC and manage it from one dashboard. Agents update themselves automatically.
